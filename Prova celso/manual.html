Manual de Uso do Site:


Página de Login:

Para acessar o site, é necessário fazer o login com seu email e senha cadastrados. Caso ainda não tenha cadastro, clique no botão "Cadastrar" para ser direcionado à página de cadastro.

Página de Cadastro:

Na página de cadastro, preencha seus dados para criar sua conta.

Página Principal:

Após fazer o login, você será direcionado para a página principal. Aqui você encontrará dois botões:

- Cadastrar: Redireciona para a página de cadastro.

- Estatística: Mostra os emails cadastrados.
